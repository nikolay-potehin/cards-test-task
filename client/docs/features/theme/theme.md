# Theme

## User story
As a user, I want to switch between light and dark themes at runtime so I can use the app comfortably in different lighting conditions.

## Actions flow
1. Launch the app.
2. Expect the app to start in the default theme.
3. Tap the theme toggle control in the top bar.
4. Expect the app to switch to the opposite theme immediately.
5. Expect the toggle icon to swap (sun <-> moon).
6. Expect all screens and cards to update their colors to match the new theme.
7. Tap the toggle again.
8. Expect the app to switch back to the previous theme and the icon to swap back.
